# Chassis, Motors, Encoders

Store chassis drawings, motor specs, wheel dimensions, encoder specs, and mechanical assembly notes here.

Critical parameters:

- Maximum vehicle size must stay within `25 cm x 15 cm x 25 cm`.
- Wheel diameter.
- Wheelbase and track width.
- Motor rated voltage/current.
- Encoder counts per revolution.
- Mounting space for line sensor and actuator.
