# Servo, Gimbal, And Actuator

Store servo/gimbal/laser/pen/marker documents here.

Baseline path:

1. Use two-axis servo gimbal for pointing.
2. Use laser dot for early aiming tests.
3. Add pen or marker only after static pointing is reliable.

Track:

- Servo supply voltage.
- Stall current.
- Mechanical angle limits.
- PWM pulse range.
- Gimbal zero position.
- Target-plane calibration points.
